# aranea_ISMS

ISMS-SystemのIoT関連開発データ

## ディレクトリ構造

```
aranea_ISMS/
├── docs/              # ドキュメント
├── code/
│   ├── ESP32/         # ESP32関連コード
│   └── Orangepi/      # Orange Pi関連コード
└── README.md
```

## 概要

このリポジトリはISMS（情報セキュリティマネジメントシステム）に関連するIoTデバイスの開発データを管理します。

### docs/
プロジェクトのドキュメント、設計書、仕様書などを格納します。

### code/ESP32/
ESP32マイコン向けのファームウェア、スケッチ、ライブラリを格納します。

### code/Orangepi/
Orange Piシングルボードコンピュータ向けのアプリケーション、スクリプト、設定ファイルを格納します。
