# Access Absorber 機能仕様

## 概要

Access Absorberは、カメラブランド/シリーズごとの接続制限を管理し、ユーザーに適切なフィードバックを提供する緩和機構です。

## 背景

### 問題
- カメラブランドによって同時接続数制限が異なる
- Tapo: 最大1接続（並列不可）
- JOOAN(NVT): 急速再接続に2秒間隔必要
- VIGI: 3並列まで安定
- ユーザーには「なんかよくわからんけど不安定」という印象を与えていた

### 解決策
- ブランド別の接続制限を明示的に管理
- 制限に達した場合はユーザーに**原因を明確に表示**
- 不安定ではなく「制限がある」ことを伝える

## 設計原則

1. **ブランド固有**: 全体適用ではなく、ブランドごとに緩和機構を適用
2. **透明性**: 制限理由をユーザーに明示
3. **グレースフル**: 制限時も代替手段を提示
4. **拡張性**: 新規ブランド追加が容易
